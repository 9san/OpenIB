About

9san.ch is a website running on OpenIB Reloaded as a result of the development for Infinity and OpenIB being permanently dead as the repository owners are no longer active and 8chan going closed source (with their embarassingly ugly design).
The fork was written by and for 9channel, but is released Open Source under a modified MIT license which can be found here.

OpenIB Reloaded is an improved and actively developed version of OpenIB which is a security fork based on Infinity which is a public board creation fork based on vichan which is an old, but maintained fork of Tinyboard which was a once upon a time cutting edge imageboard software in terms of performance, but now outdated and defunct.
